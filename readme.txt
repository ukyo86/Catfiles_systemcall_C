<概要>
OS(Linux,Windows)で、引数に指定した複数のテキストファイルの内容を結合して、一つのファイルにまとめるコマンドを作成した。
制約として、catコマンドは使用せずシステムコールを用いて実装する。
結合した結果を格納するファイルが存在しない場合はcreatシステムコールを用いて生成する。
-----------------------------------------------------------------------

<必要ファイル>
catfiles.c  引数にテキストファイルを複数指定すると、ファイルの中を結合しconcat.txtに書き写すプログラム
sample1.txt 任意のテキストファイル2つ以上用意
sample2.txt

-----------------------------------------------------------------------

コンパイル方法
gcc catfiles.c -o catfiles

実行方法(ファイルを引数にする)
.//catfiles a.txt b.txt c.txt

------------------------------------------------------------------------

各自ファイルの結合を行い、concat.txtというファイルにひとつにまとめる。
